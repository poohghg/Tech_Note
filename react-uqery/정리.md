### 데이터 변환 (Data Transformation)

#### 데이터 변환의 위치

0. 백엔드에서 진행: 데이터를 사용하는 클라이언트가 많을 경우 위 형태가 제일 좋음
1. quertFn에서 진행: 쿼리키에 매핍되는 함수에서 변환
   - 코드 전역에서 해당 키로 변환값을 조회 할 수 있다.
   - API 문서 및 네트워크 통신과정의 데이터와 일치 하지 않는다.
   - 원본 구조에 접근 할 수 없다.
