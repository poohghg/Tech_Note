> 최장 공통 부분 수열(Longest Common Subsequence, LCS)은 두 시퀀스(문자열 또는 배열)에서 공통으로 나타나는 부분 수열 중 가장 긴 것을 찾는 문제이다. 이 문제는 동적 계획법(Dynamic Programming)을 사용하여 효율적으로 해결할 수 있다.
> 
> 최장 공통 부분 수열은 두 수열이 어떤 기준에 따라 양쪽에서 공통으로 발견 할 수 있는 가장 긴 부분 수열을 의미한다.

> 컴퓨터 과학에서 수열은 특정 순서로 나열한 객체를 의미한다. 즉 원소 사이의 순서만 유지하면 되고 반드시 연속될 필요는 없다. 예를 들어, "ABCBDAB"와 "BDCAB"의 최장 공통 부분 수열은 "BCAB" 또는 "BDAB"이다.


### 점화식

#### 2가지 조건

- 두 문자열의 특정 문자가 같은지
- 같다면 찾은 두 문자의 위치가 이전에 찾은 문자의 다음에 있는지

``` ts

function longestCommonSubsequence(text1: string, text2: string): number {
    const m = text1.length;
    const n = text2.length;

    // 2차원 배열 dp 생성 및 초기화
    const dp: number[][] = Array.from({ length: m + 1 }, () => Array(n + 1).fill(0));

    // dp 배열 채우기
    for (let i = 1; i <= m; i++) {
        for (let j = 1; j <= n; j++) {
            if (text1[i - 1] === text2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1; // 문자가 같을 때
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]); // 문자가 다를 때
            }
        }
    }

    return dp[m][n]; // 최장 공통 부분 수열의 길이 반환
}
```
