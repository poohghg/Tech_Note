> 웹 페이지의 도메인 주소를 브라우저 주소창에 입력한 후 웹 페이지가 화면에 표시되기까지의 과정

### 1.도메인 주소에서 웹 서버 부조 알아내기

#### DNS 리졸브

> 브라우저는 도메인 주소를 IP 주소로 변환하기 위해 DNS(Domain Name System) 리졸버를 사용한다. DNS 리졸버는 도메인 이름을 해당하는 IP 주소로 매핑하는 역할을 한다. 브라우저는 먼저 로컬 DNS 캐시를 확인하고, 캐시에 없으면 ISP(인터넷 서비스 제공자)의 DNS 서버에 질의하여 IP 주소를 얻는다.

> HTTP가 이해하는 주소는 도메인이 아닌 IP 주소이다. 사람인 이해하기 쉬운 형태인 도메인 주소를 프로토콜이 이해하는 형태인 IP 주소로 변환해야 브라우저와 웹 서버가 서로 통신하며, 브라우저는 사용자가 입력한 도메인 주소를 IP 주소로 변환해서 웹 페이지 정보를 가진 웹 서버로 요청을 보낸다.
